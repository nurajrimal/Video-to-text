# 🎬 Video2Text — Frame-by-Frame Intelligence

Convert any video into a timestamped text transcript.

## What It Does

1. **Uploads** a video (MP4, AVI, MOV, MKV, WEBM, etc.)
2. **Extracts** frames at your chosen FPS interval using OpenCV
3. **Describes** each frame using Claude's vision model
4. **Deduplicates** similar descriptions via cosine similarity
5. **Exports** a clean timestamped `.txt` transcript

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

> If you prefer to use GPU-accelerated sentence-transformers for better semantic deduplication:
> The app falls back to lightweight word-vector cosine similarity by default — works great for deduplication.

### 2. Run the app

```bash
python app.py
```

Then open: `http://localhost:5000`

### 3. Use it

1. Enter your **Anthropic API key** (get one at https://console.anthropic.com)
2. Set **Frames per Second** — `0.5` means one frame every 2 seconds (good default)
3. Set **Similarity Threshold** — `0.85` removes near-duplicate descriptions
4. Upload your video
5. Click **Analyze Video**
6. Watch real-time progress in the terminal panel
7. Download your `.txt` transcript

## Settings Guide

| Setting | Recommended | Effect |
|---------|-------------|--------|
| FPS | 0.25–1.0 | Higher = more detail, more API calls |
| Threshold | 0.80–0.92 | Higher = stricter deduplication |

## Tips

- **Lecture/tutorial videos**: Use 0.5 FPS, threshold 0.85
- **Fast-paced content**: Use 1–2 FPS, threshold 0.80
- **Slow/static content**: Use 0.2 FPS, threshold 0.90
- **Keep videos under 10 min** for reasonable API costs

## Upgrading Deduplication (Optional)

For better semantic similarity, install sentence-transformers and update `app.py`:

```python
from sentence_transformers import SentenceTransformer
model = SentenceTransformer('all-MiniLM-L6-v2')

def get_embedding(text):
    return model.encode(text)
```

Replace the `get_simple_embedding` call with `get_embedding` in `deduplicate_descriptions()`.
